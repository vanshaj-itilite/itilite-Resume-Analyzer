How to Use the Resume Analyzer

End-to-end flow from upload to shortlist decisions.

1. Upload resumes

Use single mode for one candidate or batch mode for bulk processing. PDF files are preferred for stable extraction.

Single upload

  1. Switch to Single Analysis.
  2. Paste or upload one resume and fill role context.
  3. Save AI settings, then run analysis.

Batch upload

  1. Switch to Batch Upload.
  2. Select multiple PDFs or a folder.
  3. Run batch and track live status per file.

2. Configure AI settings

Use Global AI Settings to choose provider, model, temperature, and threshold. Save settings before running analysis.

3. Review output

Each candidate includes score + summary. In batch mode, pass/fail icons are shown against threshold and status rows track progress and final state.

4. Analyze trends

Open the Analytics page to sort historical runs by score, compare summaries, and inspect model/provider combinations used per run.
